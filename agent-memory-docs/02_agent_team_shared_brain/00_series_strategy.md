# 系列二：给 Agent 团队装一个共享大脑

## 系列定位

从单个 Coding Agent 的记忆能力，升级到多个 Agent 协作时的知识与经验共享机制。

当 Agent 进入团队协作后，核心问题不再是“如何记住”，而是：

> 如何共享、如何隔离、如何治理记忆。

## 核心问题

- 个人经验如何变成团队规则
- 不同 Agent 如何共享知识而不污染彼此
- 记忆如何分层与分权限
- 错误经验如何被阻断传播

## 四个关键能力

1. 记忆作用域（Scope）
2. 记忆可信度（Trust）
3. 记忆路由（Routing）
4. 记忆治理（Governance）

## 总体结构

```text
个人记忆 → 项目记忆 → 团队记忆 → 组织规范 → 技能库
```

## 系列目标

构建一个可治理的团队级 Agent Memory 系统，使多个 Coding Agent 能安全共享经验并持续进化。
