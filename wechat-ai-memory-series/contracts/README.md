# Article Contracts

每篇文章都必须有单独契约。契约负责约束“这一篇只讲什么、不讲什么”，防止其他 Agent 自由发挥导致系列跑偏。

当前包含：

- `ARTICLE_CONTRACT_01.md`：忘了
- `ARTICLE_CONTRACT_02.md`：误会了
- `ARTICLE_CONTRACT_03.md`：乱记了
- `ARTICLE_CONTRACT_04.md`：想不起来
- `ARTICLE_CONTRACT_05.md`：记得对不对
- `ARTICLE_CONTRACT_06.md`：什么时候该忘

写某一篇之前，必须先读对应契约，再读 Series Bible 和 Writing Skill。
