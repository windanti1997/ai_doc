# Article Contracts

每篇文章都必须有单独契约。契约负责约束“这一篇只讲什么、不讲什么”，防止其他 Agent 自由发挥导致系列跑偏。

- `ARTICLE_CONTRACT_02.md`：误会了——记住原话但没懂意思
- `ARTICLE_CONTRACT_03.md`：乱记了——临时要求变长期偏好
- `ARTICLE_CONTRACT_04.md`：想不起来——记忆有了但没触发
- `ARTICLE_CONTRACT_05.md`：记得对不对——经验必须验证
- `ARTICLE_CONTRACT_06.md`：什么时候该忘——记忆生命周期

- `ARTICLE_CONTRACT_08.md`：经验放在哪里——Memory Layers
- `ARTICLE_CONTRACT_09.md`：多个 Agent 怎么共享经验——Team Memory

- `ARTICLE_CONTRACT_12.md`：最终收束——外置大脑是成长系统

## 契约最低完整性标准

每篇契约必须包含：系列位置、标题方向、唯一核心问题、读者心声、认知推进、必写场景、禁止展开内容、推荐开头、叙事路线、结尾任务、封面提示词、跑偏风险、评审重点。

少任一项，都不能交给其他 Agent 写正文。
