# Skills

本文件夹存放给其他 Agent 使用的写作能力文件。

## 文件说明

- `WECHAT_AI_MEMORY_WRITER_SKILL_FULL.md`：当前对话迭代后的完整写作 skill，包含严格评审、标题误判反例、反评审讨好、禁止自证、金句三连、封面裁剪等硬规则。其他 Agent 写文章时优先读这个。
- `WECHAT_SERIES_WORKFLOW_SKILL.md`：完整工作流，从读取上下文到写作、评审、归档。
- `WECHAT_COVER_IMAGE_SKILL.md`：公众号封面图生成与裁剪安全区规则。
- `WECHAT_SERIES_WRITING_SKILL.md`：早期短版写作规则，保留作索引，不再作为唯一 skill 使用。

## 使用顺序

1. 先读 `context_pack/01_SERIES_BIBLE.md`；
2. 再读 `WECHAT_AI_MEMORY_WRITER_SKILL_FULL.md`；
3. 再读 `WECHAT_SERIES_WORKFLOW_SKILL.md`；
4. 写封面时读 `WECHAT_COVER_IMAGE_SKILL.md`。
