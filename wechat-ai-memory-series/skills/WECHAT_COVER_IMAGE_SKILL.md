# Skill：微信公众号封面图生成

## 固定视觉语言

本系列公众号封面默认沿用以下视觉风格：

- 轻漫画插画风；
- 浅色背景；
- 黑红大字标题；
- 黄色强调条、黄色便签、红色警示贴纸；
- 第一眼先看到大标题，第二眼看到人物冲突，第三眼看到便签短句；
- 不做纯科技蓝海报；
- 不做复杂信息流海报。

## 公众号裁剪安全区

只生成一张图，但必须兼容：

- 约 2.35:1 的消息列表横向裁剪预览；
- 1:1 的转发卡片 / 公众号主页裁剪预览。

硬规则：

1. 主标题、主人物脸、核心冲突画面、副标题必须集中在中央安全区；
2. 四周边缘只放辅助元素，如便签、贴纸、装饰线、警示符号；
3. 即使左右或上下被裁掉一部分，读者也必须还能看懂文章在讲什么；
4. 不允许把核心文案贴在最上、最下、最左、最右边缘；
5. 不要生成横版和竖版两张，必须是一张兼容多种裁剪的封面。

## 系列角色设定

- 用户角色统一使用“风逆”形象：年轻男性、短发、科技感休闲风、坐在电脑前，表情无奈、扶额或抓狂；
- AI 角色统一使用“可爱但自信过头的小机器人”；
- AI 不是邪恶角色，而是认真、积极、自信，但理解错了、用错了、套错了。

## 封面提示词模板

```text
请生成一张适合微信公众号文章封面的单张图片，延续本系列固定风格：轻漫画插画风，浅色背景，黑红大字标题，黄色强调条，黄色便签，红色警示贴纸。画面要一眼看懂冲突，适合公众号封面，吸睛但不花，不要做成纯科技蓝海报。

裁剪要求：只生成一张图，同时兼容微信公众号 2.35:1 消息列表裁剪和 1:1 转发卡片裁剪。主标题、主人物、AI、核心冲突、副标题必须集中在中央安全区。四边只放辅助元素，避免关键内容被裁掉。

封面大字：{{封面大字}}
封面副标题：{{封面副标题}}
画面构图：{{主画面冲突}}
黄色便签短句：{{便签 1}} / {{便签 2}} / {{便签 3}}

```

## 配图评审

配图评审必须检查：裁剪后主标题是否仍完整可读；裁剪后 AI 与用户是否仍能同时出现；裁剪后副标题是否仍清晰；裁剪后是否仍能一眼看懂冲突；是否延续系列视觉风格。

如果封面只在完整原图里好看，但进入公众号裁剪预览后丢失关键信息，必须判“必须改”。
